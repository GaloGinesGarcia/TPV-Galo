# TPV Laboral

## Descripción
TPV Laboral es una aplicación en C# con fines educativos que simula un terminal punto de venta (TPV) para entornos retail.

## Funcionalidades
- Gestión de productos
- Control de inventario
- Registro de ventas
- Emisión de tickets
- Gestión de clientes

- ## Tecnología
- C#
- .NET
- Programación orientada a objetos
- Base de datos (CRUD)

## Objetivo
Proyecto didáctico para aprender cómo funciona un sistema TPV en un entorno real de comercio.
